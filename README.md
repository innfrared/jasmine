# Jasmine Crafted

Jasmine Crafted is a React + TypeScript storefront application built to support a modern e-commerce experience with internationalization, modular product pages, and a scalable front-end architecture.

The application is designed to be performance-aware and structured to support SEO-oriented rendering strategies as it evolves.

## Setup

### Requirements
- Node.js 22+
- npm

### Install and run

```sh
cd client
npm install
npm run dev
```
